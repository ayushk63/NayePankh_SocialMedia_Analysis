## **NayePankh Social Media Analysis**

Analyzed Social Media Activity of Naye Pankh Foundation. Created Dashboard using Power BI and performed EDA using Python.

**Tools Used :-** Power BI, Python, Apify

!(./DashboardImage.png)

**Process :-** 

1.	Collected Data (followers, likes, comments and number of posts) from social media accounts of Naye Pankh Foundation. Collected data manually from LinkedIn, YouTube and X, and used Apify (free credits) to scrape data from Instagram and Facebook. Created dataset in Excel.
2.	Created Dashboard in Power BI having KPI’s at the top, slicers for filtering, and visual charts.
3.	Imported the dataset in Python using pandas and performed EDA using pandas and seaborn.
   
**Key Insights :-**

1.	Naye Pankh Foundation has received an average of 879 comments and 11K likes on social media platforms.
2.	It has posted an average of 126 posts on social media platforms.
3.	It has an average of 5354 followers.
   
**Recommendations :-**

1.	There are no posts on LinkedIn and X. Naye Pankh Foundation should post something related to their NGO on these platforms too.
2.	X has very low followers. Naye Pankh Foundation should try to increase their activity on X so that their followers may increase and more people may learn about the NGO and it’s activities.
